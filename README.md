# 🌍 Earthquake Data Analysis & Visualisation (2015–2025)

This project explores global earthquake activity over the past decade, combining data analysis, statistical exploration, and interactive visualisation to uncover patterns in frequency, magnitude, and depth.

The project was developed as part of a data analytics assignment and aims to bridge rigorous analysis with accessible, web-based storytelling.

---

## 📊 Project Overview

Using a dataset of ~80,000 global earthquake events, this project investigates:

- Trends in earthquake frequency over time  
- The distribution of earthquake magnitudes globally and regionally  
- Differences between global and Nepal-specific earthquake patterns  
- The relationship between earthquake depth and severity  
- Geographic clustering of seismic activity  

The analysis combines Python-based data processing with interactive visualisations for clearer insight communication.

---

## 🧠 Key Features

- **Data Cleaning & Preparation**
  - Structured preprocessing pipeline using pandas
  - Feature engineering (magnitude bins, depth categories, temporal features)

- **Exploratory Data Analysis**
  - Time-series analysis of earthquake frequency  
  - Magnitude distribution comparisons (global vs Nepal)  
  - Depth distribution and categorisation  

- **Visualisation**
  - High-quality static plots using Matplotlib  
  - Interactive maps and charts using Plotly  
  - Web-based presentation with embedded interactive elements  

- **Web Presentation**
  - Clean HTML/CSS layout
  - Interactive charts embedded via Plotly and Datawrapper  
  - Designed for intuitive storytelling and user exploration

## 📌 Key Insights

- Earthquake frequency shows moderate variability but no clear long-term trend  
- The vast majority of earthquakes fall within lower magnitude bands (M4–5)  
- Higher-magnitude events are rare but disproportionately impactful  
- Shallow earthquakes dominate globally, with important implications for damage severity  
- Regional patterns (e.g. Nepal) differ meaningfully from global distributions
